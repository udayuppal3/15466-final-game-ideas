Final Game Ideas

Designed by Uday Uppal

Game 1: Stealth Runner

Stealth Runner is a 2D puzzle/action game in which the player navigates a maze of enemies attempting to not be spotted. The game would have light and dark areas, and the player is hidden in the dark ones and exposed in the light.

Game 2: Archer

Archer is a game where the player attempts to shoot target baloons with a certain number of arrows. Each level would progressively get more complex with smaller/moving targets as well as wind influence on the arrow.

Game 3: Zombie Invasion

Zombie Invasion is a tower-defense type of game in which the player must protect a fort from an invasion of zombies. The player would get points for each enemy stopped, which they could use to build up defenses.

Game 4: Tank Attack

Tank Attack is a 3D action game in which the player progresses through levels, controlling a tank and defeating enemy tanks. The game would be played from a side view of the level, and the player would control the tank with wasd and the mouse.

Game 5: Ninja Swing

Ninja Swing is a side scrolling game in which the player controls a ninja and attemps to last as long as possible against an infinite amount of approaching enemies. The player would have ninja stars and a rope to better fight the enemies and control the ninja.

Game 6: Network Chess

Network Chess is the classic Chess game played between players on a network. Players would take turns making moves, and there would also be a time limit to keep the game going and ensure an ending.
